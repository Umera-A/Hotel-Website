# Hotel-Website
A modern and responsive hotel website built using HTML, CSS, Bootstrap 5, and JavaScript. The project showcases hotel rooms, amenities, a gallery, guest reviews, and an interactive booking interface.

*Features
 Fully Responsive Design
 Room Showcase Section
 Rooms
 Hotel Gallery
 Guest Reviews Section
 Contact Form
 Booking Modal with Payment Method Selection
 Automatic Price Calculation Based on Stay Duration
 Modern UI Built with Bootstrap and Flexbox
 
*Room Types
 Standard Room
 Deluxe Room
 Family Room

*Payment Options
The booking interface includes multiple payment choices:
   Cash on Arrival
   TeleBirr
   CBE Birr


Note: Payment processing is simulated for demonstration purposes only. No real transactions are performed.
